# BrightPath

🌟 **Your Personal Guidance to Success**

BrightPath is a responsive web application that helps you set goals, track progress, and generate QR codes for easy sharing.

## Features

✅ **Goal Setting** - Create and manage your personal goals  
✅ **Progress Tracking** - Log and monitor your achievements  
✅ **QR Code Generator** - Generate QR codes instantly  
✅ **Mobile Friendly** - Works perfectly on phones and tablets  
✅ **Desktop Compatible** - Optimized for all screen sizes  
✅ **Offline Support** - Works without internet connection  
✅ **Secure & Private** - Your data stays on your device  

## Quick Start

1. Open the website in your browser
2. Start setting your goals
3. Track your daily progress
4. Generate QR codes to share

## Deployment

Deploy this project on:
- **Netlify** - https://netlify.com
- **Vercel** - https://vercel.com
- **GitHub Pages** - Enable in repository settings

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- QRCode.js
- Service Workers (PWA)
